Task Scheduler

This is the project I worked on with my group members from CPSC 362, Software Engineering. We built a task scheduler where, a person could see encouraging quotes that were on rotate, edit a calendar with their tasks, with all their tasks showing up in the order of when they finish.

I was tasked with creating and making the calendar customizable. The basic code of the calendar came from huiwenteo.com
